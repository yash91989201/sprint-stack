# Dashboard

## Purpose

The dashboard is the personal landing surface that surfaces a user's work, recent context, and quick actions so they can jump back in without searching.

## Capabilities

- My Issues
- Assigned to Me
- Created by Me
- Recent Activity
- Upcoming Milestones
- Favorite Projects
- Quick Create
- Pinned Items
- Recently Viewed

## Notes

Dashboard widgets aggregate data from issues, projects, milestones, and views. Quick Create and Pinned Items lean on productivity features like keyboard shortcuts and pinning.
