# Activity Timeline

## Purpose

A chronological record of everything that happened to an issue, from creation to edits, comments, and metadata changes.

## Capabilities

- Issue created
- Status changed
- Assignment changes
- Priority changes
- Comments
- Label updates
- Attachments
- Milestone changes
- Complete audit history

## Notes

- Timeline events are surfaced on the issue detail page and feed the [Notifications](./notifications.md) module.
- [Comments](./comments.md), [Attachments](./attachments.md), and [Milestones](./milestones.md) all contribute events.
