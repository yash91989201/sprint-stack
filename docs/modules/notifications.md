# Notifications

## Purpose

Keep members informed about work that involves them, from assignments to replies and status changes.

## Capabilities

- Assignments
- Mentions
- Replies
- Status changes
- Milestone updates
- Project activity
- Notification preferences
- Inbox
- Read/unread
- Email notifications

## Notes

- Mention notifications are triggered by the [Rich Editor](./rich-editor.md) in [Comments](./comments.md) and issue descriptions.
- Status-change and assignment notifications are driven by the [Activity Timeline](./activity-timeline.md).
