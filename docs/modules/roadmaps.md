# Roadmaps

## Purpose

Roadmaps visualize plans over time, connecting epics, dependencies, and releases.

## Capabilities

- Timeline visualization
- Epics
- Dependencies
- Progress
- Release planning

## Notes

Roadmaps build on [Issues](issues.md), [Milestones](milestones.md), and [Views](views.md). Roadmaps are post-MVP. See [MVP scope](../mvp.md).
