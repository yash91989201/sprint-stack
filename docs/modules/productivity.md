# Productivity

## Purpose

Productivity features reduce friction for power users through keyboard control, quick actions, and bulk operations.

## Capabilities

- Keyboard shortcuts
- Command palette
- Quick create
- Bulk actions
- Multi-select
- Drag & drop
- Pinning
- Favorites
- Recently viewed

## Notes

Productivity features apply across [Issues](issues.md), [Views](views.md), [Dashboard](dashboard.md), and [Search](search.md). Keyboard shortcuts, quick create, and a basic command palette are in MVP; bulk actions, multi-select, drag-and-drop beyond core boards, and advanced pinning are post-MVP. See [MVP scope](../mvp.md).
