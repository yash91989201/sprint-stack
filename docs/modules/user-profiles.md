# User Profiles

## Purpose

User profiles provide identity and personalization for each workspace member.

## Capabilities

- Avatar
- Bio
- Timezone
- Language
- Keyboard shortcuts
- Theme preference
- Activity history

## Notes

Profiles connect to [Settings](settings.md), [Notifications](notifications.md), and [Workspaces](workspaces.md). Avatar and theme preference are in MVP; full activity history, language, and timezone are post-MVP. See [MVP scope](../mvp.md).
