# Comments

## Purpose

Issue comments let collaborators discuss work, ask questions, and share context inline. They keep conversation tied to the issue so decisions stay discoverable.

## Capabilities

- Markdown
- Replies
- Mentions
- Emoji reactions
- Edit history
- Attachments
- Code blocks
- Task lists

## Notes

- Comments appear on the issue detail page and are recorded in the [Activity Timeline](./activity-timeline.md).
- Mention a teammate to generate a [Notification](./notifications.md).
- Comment attachments use the [Attachments](./attachments.md) module.
