# Milestones

## Purpose

Milestones group issues around a target date or release so teams can plan deadlines, track progress, and see how close a body of work is to completion.

## Capabilities

- Release planning
- Deadlines
- Progress tracking
- Completion %
- Closed/open counts

## Notes

Milestones are assigned to issues and surfaced in the dashboard, filters, and views to coordinate shipping cadence.
