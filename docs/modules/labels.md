# Labels

## Purpose

Labels provide a lightweight way to categorize and tag issues so teams can quickly identify type, priority, team, or any custom dimension across a workspace or project.

## Capabilities

- Workspace labels
- Project labels
- Colors
- Icons
- Descriptions
- Usage count
- Label groups

## Notes

Labels are applied to issues and used by filters and views to group, color-code, and search work. Label groups and project labels help larger workspaces keep tag taxonomies organized.
