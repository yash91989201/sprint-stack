# Search

## Purpose

Search helps users find issues, projects, users, and comments quickly from anywhere in the workspace.

## Capabilities

- Global search
- Instant results
- Search issues
- Search projects
- Search users
- Search comments
- Saved searches
- Advanced filters

## Notes

Search works alongside [Filters](filters.md) and powers discovery across [Issues](issues.md), [Projects](projects.md), and [Comments](comments.md). Basic filters are in MVP; saved searches and advanced filters are post-MVP. See [MVP scope](../mvp.md).
