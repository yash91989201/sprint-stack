# Integrations

## Purpose
Integrations extend Sprint Stack into the rest of a team's toolchain, enabling sync with engineering, chat, billing, and automation systems.

## Capabilities

- GitHub
- Slack
- Discord
- Webhooks
- Public API

## Notes

- The Public API supports external tools and custom workflows; [API keys](settings.md) are managed in Settings.
- Webhooks can power custom automation and feed into [Notifications](notifications.md) or third-party systems.
- GitHub integration can link commits, pull requests, and issues to [Issues](issues.md).
- Slack and Discord integrations route [Notifications](notifications.md) into team channels.
- Integrations are **out of MVP scope**; see [MVP](../mvp.md).
