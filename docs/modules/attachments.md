# Attachments

## Purpose

Attach files to issues and comments so teams can share screenshots, documents, and other artifacts alongside their work.

## Capabilities

- Drag & drop
- Image preview
- PDF preview
- Downloads
- Version replacement

## Notes

- Attachments can be added from the issue detail view and from [Comments](./comments.md).
- File activity is reflected in the [Activity Timeline](./activity-timeline.md).
