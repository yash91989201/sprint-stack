# Settings

## Purpose

Settings let users customize appearance, security, notifications, and connected accounts.

## Capabilities

- Appearance
- Theme
- Notification settings
- Security
- Connected accounts
- Sessions
- API keys
- Webhooks

## Notes

Settings extend [User Profiles](user-profiles.md) and connect to [Notifications](notifications.md), [Admin](admin.md), and [Integrations](integrations.md). Appearance, theme, basic notifications, password change, and session management are in MVP; API keys and webhooks are post-MVP. See [MVP scope](../mvp.md).
