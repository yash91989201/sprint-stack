# Rich Editor

## Purpose

The shared editing surface for issue descriptions, comments, and other long-form content. It renders readable markup without requiring users to write raw Markdown.

## Capabilities

- Markdown
- Tables
- Images
- Videos
- Links
- Code blocks
- Inline code
- Quotes
- Emojis
- Slash commands
- Mentions

## Notes

- Powers descriptions in [Issues](./issues.md) and [Comments](./comments.md).
- Mentions connect to [Notifications](./notifications.md).
