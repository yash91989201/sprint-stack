# Projects

## Purpose
Projects organize issues into focused, manageable scopes. They provide visibility, health signals, and a home for project-level activity.

## Capabilities

- Unlimited projects
- Public/Private projects
- Project icon & color
- Description
- Status
- Archive/Restore
- Templates
- Overview page
- Activity feed
- Team assignment
- Project health
- Project statistics

## Notes

- Projects live inside a [Workspace](./workspaces.md).
- [Issues](./issues.md) are created within a project and inherit its visibility context.
- Project status and health rely on the [Status Workflow](./status-workflow.md).
- Project-level labels extend the workspace-wide [Labels](./labels.md) module.
